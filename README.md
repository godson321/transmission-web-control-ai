## 关于
原项目已不维护，用AI修复了一些问题和增加一些新功能
- 1、优化了大量种子时候的加载逻辑，提高相应速度
- 2、修复批量替换种子服务器无效的BUG
- 3、新增批量重命名多个种子功能
- 4、修复重命名种子时候可能无法实时刷新的问题
